## ðŸ§­ Command Line Basics

| **Command** | **Description** |
|--------------|-----------------|
| `echo output-here` | Print what is typed next to this command |
| `echo "TEXT" > file-name.txt` | Create a new file and write text inside it (overwrites if it exists) |
| `pwd` | Print the current working directory |
| `cat file-name.extension` | Print the contents of a file |
| `cat source.txt > destination.txt` | Copy the contents of one file into another |
| `ls` | List the contents of the current directory |
| `ls -R` | List all files in the current directory and subdirectories recursively |
| `cd directory-name` | Change into an existing directory |
| `cd ..` | Change backward into an existing directory |
| `cd directory-name/sub-directory-name` | Change forward on multiple directories |
| `cd ../..` | Change backward on multiple directories |
| `mkdir folder-name` | Create a new directory (folder) |
| `touch file-name.extension` | Create an empty file |
| `mv notes.txt documents/` | Move a file into another directory |
| `mv albums backup-albums` | Rename or move a directory |
| `cp source.txt copy.txt` | Copy a file |
| `cp -r folderA/ folderB/` | Copy a directory and its contents |
| `cp -r my-playlist my-playlist-copy` | Create a copy of an entire folder |
| `rm old.txt` | Delete a file |
| `rm old.txt notes.txt extra.txt` | Delete multiple files at once |
| `rmdir empty-folder/` | Remove an empty folder |
| `rmdir folder-one/ folder-two/ folder-three/` | Remove multiple empty folders |
| `rm -r folder-to-delete/ source.txt destination.txt` | Delete a folder and its contents (recursively) |
| `clear` | Clear command line |
